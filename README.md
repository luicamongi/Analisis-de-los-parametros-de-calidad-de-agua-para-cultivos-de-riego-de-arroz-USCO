# An-lisis-de-los-par-metros-de-calidad-de-agua-para-cultivos-de-riego-de-arroz-USCO
En el presente trabajo se analizan los parámetros de calidad del agua asociados al cultivo de arroz mediante estrategias de agricultura de precisión, con el objetivo de evaluar su impacto sobre el recurso hídrico y apoyar la toma de decisiones en el manejo agrícola.
